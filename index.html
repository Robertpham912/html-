📘 Góc học lập trình

1. Scratch là gì? – Ngôn ngữ lập trình kéo thả rất dễ và vui! Mình đã làm nhiều game trên đó.

2. HTML là gì? – Là ngôn ngữ tạo ra các trang web. Mình dùng HTML để viết trang web đầu tiên nè!

3. Tại sao nên học sớm? – Vì lập trình giúp mình sáng tạo, tư duy logic và làm ra thứ “ảo diệu”.

4. Dùng iPad học lập trình? – Mình dùng Safari, VS Code Web, GitHub… và bàn phím là đủ chiến rồi! 😎

5. Mục tiêu của mình: – Mỗi ngày học một ít để sau này trở thành lập trình viên siêu xịn! 💻🚀

📚 Các bài học lập trình

🔸 HTML cơ bản | 🔸 CSS dễ hiểu | 🔸 JavaScript nhập môn

🔸 HTML cơ bản

HTML là ngôn ngữ đánh dấu giúp tạo ra trang web. Ví dụ: <h1>Tiêu đề</h1>

🔸 CSS dễ hiểu

CSS giúp làm cho web đẹp hơn bằng cách thay đổi màu sắc, chữ, khoảng cách...

🔸 JavaScript nhập môn

JavaScript làm trang web thông minh hơn, như xử lý nút bấm, trò chơi, hiệu ứng chữ gõ...
