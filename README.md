# html-hello
Admin Commands 
;dark
Bật chế độ tối (Dark mode) 🌙
;light
Bật chế độ sáng (Light mode) ☀️
;music on
Bật nhạc nền 🔊
;music off
Tắt nhạc nền 🔇
;bg [màu]
Đổi màu nền (ví dụ: ;bg red, ;bg #00ffcc) 🎨
;reset bg
Trả về nền mặc định 🌈
;hide [id]
Ẩn phần tử có id tương ứng (ví dụ: ;hide clock-container) ❌
;show [id]
Hiện phần tử có id tương ứng ✅
